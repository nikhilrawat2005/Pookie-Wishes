✨ POOKIE WISHES - TEMPLATE PERSONALIZATION GUIDE ✨
==================================================

This document lists all the fields required from users to personalize their surprise templates.
Fields marked as (Optional) will use premium defaults if left empty.

--------------------------------------------------
🎀 HELLO KITTY (hello-kitty)
--------------------------------------------------
- Birthday Person's Name: Their name or nickname
- Your Name (Sender): How you want to sign the surprise
- Personal Birthday Letter: A heartfelt message (shown in flip cards)
- 3 Favorite Photos: Memories for the animated gallery
- QR Code Theme (Optional): Default is "Romantic Red" (Kawaii Pink style)

--------------------------------------------------
⚡ HARRY POTTER (harry-potter)
--------------------------------------------------
- Witch / Wizard's Name: Name for the official Hogwarts letter
- Your Name (Sender): Who is sending this magical surprise
- Birthday Wish Message: Heartfelt note revealed at the end
- 3 Favorite Photos: Memories for the enchanted photo gallery
- QR Code Theme (Optional): Default is "Magical Red" (Gryffindor style)

--------------------------------------------------
💘 LOVE TRAP (love-trap)
